# **App Name**: Glycemia Tracker

## Core Features:

- Add Entry Form: Form to add new blood glucose entries with fields for date, time, meal type, glycemia (g/L), and Novorapide dosage.
- Entry List View: Display a scrollable list of past entries, showing key details for each log.
- Edit Entry: Enable editing of existing log entries to correct or update information.
- Pattern Recognition and Suggestion: Analyze historical blood glucose data to identify patterns and suggest potential adjustments to Novorapide dosage as a tool to better control blood sugar levels.

## Style Guidelines:

- Primary color: Soft, muted blue (#64B5F6) to evoke calmness and trust.
- Background color: Light gray (#F0F4F8) for a clean, readable surface.
- Accent color: Deeper blue (#1976D2) to highlight interactive elements and important information.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable text.
- Simple, clean line icons for key actions and categories. Consider using icons from the Material Design library.
- Use a card-based layout to organize data entries in a clear, visually distinct manner.
- Subtle transitions and animations for actions such as adding or editing entries, providing smooth feedback to the user.
